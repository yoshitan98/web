陳謝 - NifToEml 旧 ver. 1.0で 2000年代に1900年代の日付メッセージを処理すると *.eml の日付が2000年代になってしまいます。

< FixNTE >


作成日: 2000年1月13日

作成者: Yoshitan (mailto:yoshitan@imasy.or.jp)

動作環境: Windows 95/98 日本語版


Y2K問題のある NifToEml 旧版で Outlook Express にインポート済
みのメッセージの日付（年）を修正します（元ログファイルからの
再処理が面倒な場合に利用できます）。


使用方法

1. 任意のフォルダをハードディスク上に作成し、Outlook Express
から書き出した *.eml ファイルをその中に入れる。

2. 本プログラムを実行し、そのフォルダを指定して OK ボタンを
クリックする。

3. 修正が必要なファイルだけ修正される。

4. Outlook Express に *.eml をドラッグ&ドロップする。
